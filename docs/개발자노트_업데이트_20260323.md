## 4. 운영 체크포인트
1. 가동률 저장 실패 시 오류 노출이 즉시 이뤄지는지
2. 표준품셈 목록/반영 데이터 누락이 없는지
3. ESC 취소, Undo/Redo, 선택행 하단 추가 동작이 정상인지
4. 준비/마감 `전체기간` 입력 시 합계와 간트가 동일 기준으로 갱신되는지
