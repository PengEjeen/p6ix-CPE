# 공기검토플랫폼 사용자 가이드

작성일: 2026-03-04  
대상: 공기검토플랫폼 웹 사용자

이 문서는 "기능 목록"이 아니라, **처음 하는 사람이 실제로 끝내는 순서(업무 흐름)**로 안내합니다.

---

## 전체 공기산정 프로젝트

### Quick Start (10분) · "스케줄 입력 → 저장 → 내보내기"

1. 사이드바에서 "새 갑지"를 누르고, 프로젝트 유형을 "전체 공기산정"으로 선택한 뒤 프로젝트를 생성하세요.
2. 상단 헤더 드롭다운에서 "가동률"로 이동해 지역/기간/주간 작업일 등을 확인하고 "저장"하세요. (처음엔 기본값 그대로 저장해도 됩니다)
3. 상단 헤더 드롭다운에서 "공사기간 산정 기준"으로 이동하세요.
4. 테이블에서 공정(작업 항목), 수량, 생산성, 작업인원을 입력하세요.
5. 상단의 "저장"을 누르세요.
6. "엑셀 내보내기"로 파일을 내려받아 확인하세요.
7. (선택) 상단 헤더의 "보고서 내보내기"로 보고서를 내려받으세요.

### 1) 프로젝트 만들기(왼쪽 사이드바)

#### 이 페이지에서 하는 일

프로젝트를 만들고, 최근 작업하던 프로젝트를 다시 열어 계속 작업합니다.

#### 따라하기

1. 왼쪽 사이드바에서 "새 갑지"를 누르세요.
2. 프로젝트명을 입력하세요.
3. 프로젝트 유형에서 "전체 공기산정"을 선택하세요.
4. "생성"을 누르세요.
5. 생성된 프로젝트를 목록에서 클릭해 진입하세요.

#### 자주 막히는 점/주의

- 프로젝트 목록이 안 보이면: 사이드바가 접혀있는지 확인하세요.
- 프로젝트가 열렸는데 메뉴가 안 뜨면: 상단 헤더에 프로젝트명이 표시되는지 확인하세요.

### 2) 가동률

#### 이 페이지에서 하는 일

날씨/휴일/근무조건을 기준으로 **가동률(%)**을 계산하고 저장합니다. 저장된 값은 "공사기간 산정 기준"의 기간 계산에 영향을 줍니다.

#### 따라하기

1. (프로젝트 진입 후) 상단 헤더 드롭다운에서 "가동률"을 선택하세요.
2. 상단 설정에서 지역(관측소)과 데이터 기간(최근 N년)을 선택하세요.
3. "공공/민간", "주간 작업일(주 5/6/7일)"을 확인/선택하세요.
4. 동절기/혹서기/강우/강설 등 기준이 필요하면 값을 조정하세요. (모르면 기본값 유지)
5. 화면 하단의 산출 결과(작업일/불능일/휴일/가동률%)를 확인하세요.
6. "저장"을 누르세요.

#### 자주 막히는 점/주의

- 저장했는데 스케줄 기간이 안 바뀌면: 스케줄 화면으로 이동 후 값을 다시 확인하세요(새로고침이 필요할 수 있습니다).
- 어떤 항목을 바꿔야 할지 모르겠으면: 지역/주간 작업일만 먼저 맞추고 저장하면 됩니다.

### 3) 표준품셈

#### 이 페이지에서 하는 일

스케줄에서 가져다 쓸 **표준 생산성 데이터**를 추가/수정합니다. "공사기간 산정 기준"에서 빠르게 항목을 채우는 데 도움이 됩니다.

#### 따라하기

1. 상단 헤더 드롭다운에서 "표준품셈"으로 이동하세요.
2. 필요한 분류(대공종/공정)를 펼치세요.
3. 값을 수정했다면 "저장"을 누르세요.
4. 새 항목이 필요하면 "추가(+)"로 항목을 만드세요.
5. "공사기간 산정 기준"으로 돌아가 표준 항목을 가져오세요.

#### 자주 막히는 점/주의

- 표준값을 수정하면: 같은 프로젝트에서 스케줄에 가져오는 값이 달라질 수 있습니다.

### 4) 생산성 근거(필요한 경우만)

#### 이 페이지에서 하는 일

말뚝/CIP 등 공법별 조건을 넣고, **일일 생산성(근거 데이터)**을 계산해 확인합니다.

#### 따라하기

1. 상단 헤더 드롭다운에서 해당 페이지("CIP 생산성 근거", "기성말뚝 생산성 근거", "현장타설말뚝 생산성 근거")로 이동하세요.
2. 직경/공법/지층/깊이 등 조건을 입력하세요.
3. 계산 결과(소요시간/일일 생산성)를 확인하세요.
4. "저장"을 누르세요.
5. 필요한 경우, 결과를 참고해 스케줄의 생산성 값을 입력하거나 표준 데이터로 반영하세요.

#### 자주 막히는 점/주의

- 입력값을 지웠는데 계산이 이상하면: 필수 조건(직경/공법 등)이 비어있는지 확인하세요.

### 5) 공사기간 산정 기준(테이블 + 간트)

#### 이 페이지에서 하는 일

공정 항목을 입력해 **전체 공사기간을 계산**하고, 테이블/간트에서 흐름을 확인한 뒤 저장·내보내기 합니다.

#### 따라하기

1. 상단 헤더 드롭다운에서 "공사기간 산정 기준"으로 이동하세요.
2. 필요하면 상단에서 "Start Date"를 설정하세요.
3. 테이블에서 공정(작업 항목), 단위/수량, 생산성, 작업인원을 입력하세요.
4. 기간(일)이 자동으로 계산되는지 확인하세요.
5. 공정 순서를 바꾸려면 행의 드래그 핸들을 잡고 이동하세요.
6. 간트차트에서 막대를 이동/늘려 기간을 조정하세요. (기간을 바꾸면 인원/생산성 값이 함께 바뀔 수 있습니다)
7. 선후행 관계가 필요하면 간트에서 공정 간 연결을 추가하세요.
8. 상단의 "저장"을 누르세요.

#### 자주 막히는 점/주의

- 기간이 0으로 나오면: 수량/생산성/작업인원이 비어있거나 0인지 확인하세요.
- 저장 전에는: 새로고침/이동 시 변경이 사라질 수 있으니 "저장"을 먼저 누르세요.

### 6) 목표 공기 맞추기(AI) · 선택

#### 이 페이지에서 하는 일

목표 공기(일)를 넣으면, 시스템이 **기간을 줄이기 위한 조정안**을 보여줍니다. 마음에 들면 적용하고, 아니면 취소하세요.

#### 따라하기

1. "공사기간 산정 기준" 화면으로 이동하세요.
2. 상단의 "목표 공기"에 목표 일수를 입력하세요.
3. "기간 조정"을 누르세요.
4. 오른쪽 "AI 조정 로그"에서 변경 내용을 확인하세요.
5. 적용하려면 "AI 조정안 적용"을 누르세요.
6. 그만두려면 "취소/되돌리기"를 누르세요.

#### 자주 막히는 점/주의

- 결과가 바로 안 보이면: 계산이 끝나면 화면에 표시됩니다.

### 7) 내보내기(엑셀/보고서)

#### 이 페이지에서 하는 일

스케줄을 엑셀로 내보내거나, 보고서를 내려받습니다.

#### 따라하기

1. "공사기간 산정 기준" 화면으로 이동하세요.
2. "엑셀 내보내기"를 눌러 파일을 내려받으세요.
3. (전체 공기산정 프로젝트) 상단 헤더에서 "보고서 내보내기"를 눌러 보고서를 내려받으세요.

#### 자주 막히는 점/주의

- 다운로드가 안 되면: 브라우저 다운로드 차단 여부를 확인하세요.

## 공기 계산 프로젝트

### Quick Start (10분) · "입력 → 자동 저장 → 결과 확인"

1. 사이드바에서 "새 갑지"를 누르고, 프로젝트 유형을 "공기 계산"으로 선택한 뒤 프로젝트를 생성하세요.
2. 상단 헤더 드롭다운에서 "공기산정"으로 이동하세요.
3. "공사개요"에서 층수/주변환경 등 기본값을 입력하세요.
4. "근무조건 및 가동률"에서 주간 근무일을 선택하고, 필요하면 "직접입력(%)"을 켜서 값을 넣으세요.
5. "토공사"와 "골조공사"를 순서대로 입력하세요.
6. 화면 상단에 "업데이트 중..."이 보이면 저장 중이니 잠시 기다리세요.

### 1) 프로젝트 만들기(사이드바)

#### 이 페이지에서 하는 일

프로젝트를 만들고, 계속 작업할 프로젝트를 선택합니다.

#### 따라하기

1. 왼쪽 사이드바에서 "새 갑지"를 누르세요.
2. 프로젝트명을 입력하세요.
3. 프로젝트 유형에서 "공기 계산"을 선택하세요.
4. "생성"을 누르세요.
5. 생성된 프로젝트를 목록에서 클릭해 진입하세요.

#### 자주 막히는 점/주의

- 저장 중에는: 페이지 이동이 제한될 수 있습니다. "업데이트 중..." 표시가 사라질 때까지 기다리세요.

### 2) 공기산정

#### 이 페이지에서 하는 일

공사개요/근무조건/토공사/골조공사/공사기간을 입력하면, 결과가 자동으로 반영됩니다.

#### 따라하기

1. 상단 헤더 드롭다운에서 "공기산정"으로 이동하세요.
2. "공사개요"에 필요한 값을 입력하세요.
3. "근무조건 및 가동률"에서 주간 근무일(주 5/6/7일)을 선택하세요.
4. 필요하면 "직접입력(%)"을 켜고 값을 입력하세요.
5. "토공사"를 입력하세요.
6. "골조공사"를 입력하세요.
7. "공사기간"에서 준비/정리/마감 등 기간을 확인/입력하세요.

#### 자주 막히는 점/주의

- 저장이 느리면: 입력 후 잠시 기다리면 자동으로 반영됩니다.

### 3) 적용기준 · 선택

#### 이 페이지에서 하는 일

프로젝트에 적용할 기준값(준비·정리·마감, 토공사, 골조공사)을 미리 설정해두면 입력이 쉬워집니다.

#### 따라하기

1. 상단 헤더 드롭다운에서 "적용기준"으로 이동하세요.
2. "준비·정리·가설·마감공사" 기준을 확인/수정하세요.
3. "토공사", "골조공사" 기준을 확인/수정하세요.
4. 우측 상단의 "저장"을 누르세요.
5. "공기산정"으로 돌아가 입력을 이어가세요.

#### 자주 막히는 점/주의

- 기준을 바꾸면: 이후 계산 결과가 달라질 수 있습니다.

### 4) 가동률 · 필요할 때만

#### 이 페이지에서 하는 일

가동률(%) 산정 기준을 설정하고 저장합니다. 공기 계산 프로젝트에서는 "공기산정" 화면의 근무조건/직접입력과 함께 사용될 수 있습니다.

#### 따라하기

1. 상단 헤더 드롭다운에서 "가동률"로 이동하세요.
2. 지역/기간/주간 작업일 등을 확인하세요.
3. "저장"을 누르세요.
4. "공기산정"으로 돌아가 결과를 확인하세요.

#### 자주 막히는 점/주의

- 어떤 값을 써야 할지 모르겠으면: "공기산정" 화면의 "직접입력(%)"을 우선 사용하세요(현장 값이 있을 때).

## 공통(계정/화면)

### 로그인

#### 이 페이지에서 하는 일

아이디/비밀번호로 로그인합니다.

#### 따라하기

1. 로그인 화면에서 아이디/비밀번호를 입력하세요.
2. "로그인"을 누르세요.
3. (표시되는 경우) Google 로그인 버튼을 사용할 수 있습니다.
4. 로그인 후 홈 화면이 보이면 완료입니다.
5. 로그아웃은 사이드바 하단 사용자 영역에서 "로그아웃"을 누르세요.

#### 자주 막히는 점/주의

- 로그인에 실패하면: 아이디/비밀번호를 다시 확인하세요.

### 내 정보

#### 이 페이지에서 하는 일

내 정보 확인 및 비밀번호 변경을 합니다.

#### 따라하기

1. 사이드바 하단 사용자 영역을 펼치세요.
2. "내 정보"를 누르세요.
3. 비밀번호를 바꾸려면 안내에 따라 현재 비밀번호를 확인한 뒤 새 비밀번호를 입력하세요.
4. "저장" 또는 "변경" 버튼을 누르세요.
5. 완료 메시지를 확인하세요.

#### 자주 막히는 점/주의

- 소셜 로그인 계정은 비밀번호 변경이 제한될 수 있습니다.
